# Security Policy

## Supported Versions

Only the latest released version is supported.

## Reporting A Vulnerability

Please do not publish security issues publicly before they are reviewed.

Open a private security advisory on GitHub if available, or create an issue with a minimal description and mark it as security-related. Avoid posting sensitive files or personal documents.

## Scope

MD_View-Edit_macOS is a local macOS app for opening Markdown and text files. It should not require network access for normal use.
